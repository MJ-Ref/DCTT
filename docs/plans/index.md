## Plan Template

When starting new work, create a plan in `active/` with:

```markdown
# Plan Name

**Status:** In Progress
**Started:** YYYY-MM-DD
**Goal:** One sentence

## Steps

1. [ ] Step one
2. [ ] Step two

## Decision Log

| Date | Decision | Rationale |
|------|----------|-----------|
```

Move to `completed/` when done.
